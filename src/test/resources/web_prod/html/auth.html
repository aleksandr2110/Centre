Фритюрниця електрична професійна GoodFood EF6

У фритюрницях серії EF торгової марки GoodFood використано міжнародний популярний дизайн, корпус виконаний з харчової нержавіючої сталі. Вироби оснащені високоякісними електричними елементами, вони є енергозберігаючими та безпечними. Апарат зручний, економічний та легко чиститься. Фритюрниці серії EF мають високу якість і застосовуються для смаження різних продуктів: картопля фрі, пончики, чебуреки. Є найкращим вибором для закладів громадського харчування, готелів, ресторанів, кафе, фастфуду тощо. Можна використовувати приготування чебуреків, т.к. ТЕН захищений металевим кожухом.
